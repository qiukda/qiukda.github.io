Windows环境使用hugo搭建Github博客

2022-01-15

Windows系统中使用hugo、git、Typora工具以及Github搭建自己的博客。

创建Github仓库

在Github上创建自己的仓库，仓库名为[账户名].github.io备用。http://[账户名].github.io/即为最后你博客的地址。

记录下仓库路径

1

https://github.com/[账户名]/[账户名].github.io.git

安装软件

下载hugo二进制文件，之后在系统环境变量中添加该二进制文件所在路径 https://github.com/spf13/hugo/releases

安装git二进制工具 https://www.git-scm.com/download/win

安装Typora https://typora.com.cn/

创建本地站点

在git工具终端中输入hugo new site [path]创建站点 选择主题：https://themes.gohugo.io/

我选择的even主题

1

git clone https://github.com/olOwOlo/hugo-theme-even themes/even

将even目录拷贝到站点路径[path]的themes路径下。将even/exampleSite/config.toml拷出到[path]，按照注释进行配置。

新建博客

可在[path]根目录下创建Shell脚本，在Git命令行工具中使用，作用是创建博客，并唤醒本地的Typora工具编辑该博客。 add.sh

1
2
3
4
5

current_path=`pwd`
file_path=${current_path}/content/post/${1}
typora_path="D:/Typora/Typora.exe"
hugo new post/${1}
${typora_path} ${file_path}

使用方式：bash add.sh [博客名].md 将draft的值改为false，按照even/exampleSite路径下的示例进行配置。

首次关联Github

可在[path]根目录下创建Shell脚本，在Git命令行工具中使用，作用是创建Github上仓库的本地分支，用户后续站点的上传。 init.sh

1
2
3
4

hugo --theme=even --baseUrl="http://[账户名].github.io/"
cd public
git init
git remote add origin https://github.com/[账户名][账户名].github.io.git

编译本地站点并上传Github

可在[path]根目录下创建Shell脚本，在Git命令行工具中使用，作用是编译并上传本地站点到Github。 upload.sh

1
2
3
4
5

hugo --theme=even --baseUrl="http://[账户名].github.io/"
cd public
git add -A
git commit -m "${1}"
git push -u origin master
